## 📖 Sobre o Projeto

O **AgendaiFisio** é um sistema desenvolvido com o objetivo de facilitar o agendamento e o gerenciamento de clínicas de fisioterapia. Este projeto é resultado do esforço e dedicação de alunos durante a Sprint 1.

> **Nota Institucional:** Este projeto foi desenvolvido por alunos da **Universidade Paulista (UNIP) - Campus Ribeirão Preto** como parte das atividades acadêmicas.

## 🚀 Tecnologias Utilizadas

- **Back-End:** C#, .NET, Entity Framework, MySQL
- **Front-End:** HTML, CSS, JavaScript 

## ⚙️ Como Executar

### 🛠️ Back-End
1. Acesse o diretório da API:
   ```bash
   cd src/Clinica.API
   # ou cd clinicaproject/src/Clinica.API
   ```
2. Restaure os pacotes do projeto:
   ```bash
   dotnet restore
   ```
3. Crie e atualize o banco de dados MySQL:
   ```bash
   dotnet ef database update
   ```
4. Inicie a API:
   ```bash
   dotnet run
   ```

### 🌐 Front-End
1. Navegue até a pasta do Front-End (ou abra o diretório no VS Code):
   ```bash
   cd src/Clinica.Web
   # ou cd clinicaproject/src/Clinica.Web
   ```
2. Abra o arquivo `index.html` utilizando a extensão **Live Server** no VS Code.
